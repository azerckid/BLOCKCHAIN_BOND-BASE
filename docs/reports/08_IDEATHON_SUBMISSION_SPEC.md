# [기획/보고 08] 아이디어톤 제출용 프로젝트 명세서 (Submission Spec)

## 1. 프로젝트 기본 정보
- **프로젝트명**: BondBase x ChoonSim (춘심)
- **분야**: Web3 / RWA (Real World Asset) / AI
- **핵심 키워드**: IP 수익권 토큰화, Creditcoin 3.0, 복리 재투자(Auto-Compound), AI Intelligence

## 2. 프로젝트 퍼포먼스 (Key Highlights)
### 2.1 춘심 (ChoonSim) IP의 강력한 자산성
- 글로벌 숏폼 및 SNS 통합 팬덤 보유.
- 실물 매출(MD, 라이선싱, AI 서비스 등)이 발생하는 검증된 IP를 온체인 자산으로 수용.

### 2.2 기술적 차별점 (The Edge)
1. **Universal Oracle 기반 데이터 검증**: 가짜 매출이 아닌, 실물 API 데이터를 오라클이 정기적으로 긁어와 온체인 수익(Yield)으로 확정하는 안정적인 정산 시스템.
2. **신뢰 기반의 유동성 순환**: USDC 수익을 다시 IP 지분 채권(CHNSM Token)으로 즉시 전환하는 '자동 복리 재투자' 엔진 구현.
3. **Remix & Edge Runtime 최적화**: 전 세계 어디서든 지연 없는 금융 대시보드 경험 제공.

## 3. 사용자 여정 (User Journey)
1. **Discovery**: 투자자는 BondBase 마켓에서 활성화된 춘심 IP 채권의 성과와 예상 수익률(APR)을 확인합니다.
2. **Subscription**: USDC를 통해 IP 수익권을 구독하고 ERC-1155 기반의 지분 토큰을 수령합니다.
3. **Real-time Audit**: 시스템은 24시간 주기로 매출 데이터를 검증하여 투자자의 대시보드에 수익을 누적합니다.
4. **Growth Loop**: 투자자는 수익을 현금화(Claim)하거나, 더 큰 미래 가치를 위해 재투자(Reinvest)하여 지분율을 높입니다.

## 4. 사회적/경제적 임팩트
- **IP 민주화**: 거대 자본만 소유하던 IP 수익권을 개인 팬들에게 투명하게 분배.
- **RWA 활성화**: 실물 경제 데이터와 블록체인 스마트 컨트랙트를 결합한 실용적 금융 모델 제시.
