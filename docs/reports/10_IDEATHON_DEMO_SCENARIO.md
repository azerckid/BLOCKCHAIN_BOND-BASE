# [기여 11] 아이디어톤 라이브 데모 시나리오 (Demo Storyboard)

## 0. 시작 환경
- **준비**: BondBase 포트폴리오 대시보드 화면이 띄워져 있음.
- **상태**: 현재 CHNSM(춘심 채권) 1,000개 보유 중, 미수령 수익 50 USDC 노출.

## 1단계: 수익 확인 (Awareness)
- **사용자 질문**: "춘심아, 나 오늘 정산된 수익이 얼마나 돼?"
- **춘심 AI 답변**: "주인님! 오늘 춘심이의 글로벌 이모티콘 판매가 대박 났어요! 🚀 현재 주인님의 지분에 따라 **50.25 USDC**가 정산되어 누적됐어요. 실시간 온체인 오라클로 방금 확인 완료했답니다!"

## 2단계: 자동 재투자 실행 (Growth)
- **사용자 질문**: "오, 잘나오네! 이 수익을 다시 춘심이한테 투자해주면 어떻게 돼?"
- **춘심 AI 답변**: "탁월한 선택이에요! 지금 바로 수익금을 재투자하면 복리 엔진이 작동해서 주인님의 채권 지분이 약 **52개** 더 늘어나요. 1년 뒤엔 지분이 **1.5배**가 될 수도 있다고요! 지금 바로 재투자할까요?"
- **사용자**: "응, 진행해 줘."
- **효과**: 화면상의 CHNSM 토큰 개수가 1,000 -> 1,052로 업데이트되는 애니메이션 노출.

## 3단계: 임팩트 확인 (Impact)
- **사용자 질문**: "투자한 돈이 어떻게 쓰이고 있는지 보여줘."
- **춘심 AI 답변**: "주인님의 소중한 투자금은 이번 춘심이의 도쿄 팝업스토어 홍보 비용으로 쓰이고 있어요. 덕분에 일본 팬들이 **300%** 더 늘어났답니다! 📈 여기 현재 유동성 영향력 지도가 보이시죠?"
- **효과**: 대시보드의 'Impact Map' 활성화.

## 4단계: 마무리
- **멘트**: "이처럼 춘심 AI는 복잡한 온체인 데이터를 투자자의 언어로 번역하고, 투자가 성장이 되는 과정을 실시간으로 체감하게 해줍니다."
