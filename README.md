# BondBase x ChoonSim: Next-Gen RWA Launchpad 🚀

**An innovative IP Yield Tokenization platform leveraging the "Raise-to-Earn" model on Creditcoin 3.0.**

![BondBase x ChoonSim](https://via.placeholder.com/800x400?text=BondBase+x+ChoonSim)

## 📖 Project Overview

**BondBase x ChoonSim** is a next-generation Real World Asset (RWA) launchpad that tokenizes the revenue rights of the globally recognized short-form IP, **"ChoonSim."** By pegging the IP's real-world revenue (from MD, licensing, AI services, etc.) to USDC and tokenizing it via the **BondBase RWA Protocol** on **Creditcoin 3.0**, we create a transparent and engaging investment ecosystem.

Our platform introduces the **"Raise-to-Earn"** model: Investors provide liquidity to help the IP grow, and in return, they earn yields directly correlated with the IP's real-world success. This symbiotic relationship provides IPs with fandom-based liquidity and investors with emotional and financial returns.

## ✨ Key Highlights & Features

### 1. The "Raise-to-Earn" Ecosystem
- Invest in your favorite IP's growth and earn tangible USDC yields.
- A seamless transition from being a passive fan to an active stakeholder.

### 2. Creditcoin 3.0 & Universal Oracle Integration
- **Creditcoin 3.0 (CC3):** Built on the robust CC3 Testnet for secure, transparent, and scalable transactions.
- **Universal Oracle Data Verification:** Real-world API data (sales, ad revenue) is constantly audited and verified by the oracle to finalize on-chain yields reliably.

### 3. Auto-Compound & Fractional Ownership
- **ERC-1155 Tokenization:** Flexible and efficient IP share distribution.
- **Auto-Compound Engine:** Optionally reinvest your USDC yields instantly back into "CHNSM" token shares to effortlessly maximize your holdings.
- **Growth Loop:** Claim yields for liquidity or reinvest to expand ownership.

## 🛠 Tech Stack

### Blockchain & Smart Contracts
- **Network:** Creditcoin 3.0 (CC3) Testnet

- **Framework:** React Router v7 (Vite)

### Backend & AI Intelligence
- **Database:** Turso (libSQL) / Drizzle ORM

- **AI Engine:** Google Gemini 2.5 Flash (Primary) / OpenAI GPT-4O (Fallback) via Vercel AI SDK

## 🚀 User Journey

1. **Discovery:** Browse the BondBase market to check ChoonSim IP's performance metrics and APR.
2. **Subscription:** Subscribe to IP revenue rights using USDC and receive ERC-1155 share tokens.
3. **Real-time Audit:** Watch as the oracle verifies real-world revenue data every 24 hours, automatically accumulating your yield.

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/bondbase-choonsim.git
   cd bondbase-choonsim
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.development` file in the root based on `.env.example` and add your Turso, Vercel, and Web3 credentials.

4. Start the development server:
   ```bash
   npm run dev
   ```
